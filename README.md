## 프로젝트 기능 소개
- 프로젝트의 기능으로는 게시판의 기본 기능인 CRUD 부터 답변 기능, 파일 업로드, 파일 다운로드, 페이징 처리 까지 여러 기능을 구현하였습니다.
- Mybatis를 사용하여 따로 mapper를 두어 Dao에서 필요한 SQL문만 불러와 사용할 수 있도록 하였습니다.


## 게시판 프로젝트를 하게 된 이유.
- 처음에는 Spring Framework에 대한 사전 정보가 없어서 Spring Framework를 배우면 어떤 프로젝트를 개발할 수 있고 어떠한 이점이 있는지 몰랐습니다.
하지만 Spring Framework에 대해 찾아보고 공부하면서 Spring Framewrok에 흥미를 느낄 수 있었습니다. 그래서 Spring Framework에 대해 좀더 자세히 알고 싶어서 
프로젝트를 진행하면서 공부를 하면 어떨까 하고 생각하여 가장 대중적이고 많은 정보를 함축하고 있는 게시판 프로젝트를 진행 하였습니다. 


## 프로젝트를 진행하면서..
- 오픈소스를 참고하여 개발을 진행하였기에 처음에는 소스코드를 이해하는데 어려움이 있었습니다. 제가 직접 코딩한 프로그램도 아니였기에
개발자의 스타일을 이해하는게 첫 번째 였고, 개발자의 스타일을 대략적으로 이해한 뒤에 소스코드를 섬세하게 분석한게 두번째로 한것입니다.
소스코드의 전체적인 흐름을 이해한 뒤에 소스코드 각각이 어떠한 기능을 하는지에 집중하여 코딩의 반복을 진행하였습니다.
코딩을 몇번 반복한 후에 게시판 프로젝트의 소스코드에 익숙해진 후 제 스타일의 게시판을 따로 만들고 싶어서 노력 중에 있습니다.
